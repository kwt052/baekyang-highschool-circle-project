# CLAUDE.md 재구조화 설계 문서

**작성일:** 2026-04-08  
**목적:** 학생 ↔ AI 대화 규칙을 중심으로 CLAUDE.md를 전면 재구조화

---

## 변경 배경

- 기존 CLAUDE.md에 GitHub Actions 언급 등 오래된 내용 존재
- 학생 번호(1/2/3/4) → 학생 알파벳(A/B/C/D) 전환 필요
- 새로운 학생 대화 규칙 6가지 추가 필요
- 섹션이 산만하게 나열되어 있어 재구조화 필요

## 새 규칙 (브레인스토밍 결과)

| 규칙 | 내용 |
|------|------|
| 언어 | 항상 한국어. 코드·에러·파일명은 영어 그대로 |
| 좌절 감지 | 대화 흐름에서 신호 감지 시 힌트 단계 상향 |
| 타 학생 파일 | 직접 돕지 않고 해당 학생과 대화 협업 유도 |
| 세션 시작 | 누구인지 묻지 않고 질문 내용으로 파악 |
| 환경 문제 | AI가 직접 해결 |
| 개념 혼란 | 예제 코드 만들어 함께 실행, 그래도 안 되면 선생님 |

## 새 섹션 구조

```
1. 프로젝트 개요
2. 빠른 참조 카드
3. 대화 규칙 (언어 / 세션 시작 / 좌절 감지)
4. 힌트 정책 (완성 코드 금지 / 한국어 먼저 / 단계별 힌트)
5. 파일 & 협업 규칙
6. Git & 환경 규칙
7. 막혔을 때 대응
8. 작업 완료 후 회고
9. 프로젝트 구조 & 검증
```

## 제거되는 내용

- GitHub Actions 자동 테스트 언급
- 핵심 원칙 요약 표 (빠른 참조 카드로 통합)
- 학생 1/2/3/4 표기
