## 2. 프로젝트: 백양고 퀴즈봇

터미널에서 실행하는 대화형 퀴즈 프로그램. 완성 후 4명이 함께 퀴즈를 풀면서 서로 알아갈 수 있습니다.

### 파일 구조

```
baekyang-highschool-circle-project/
├── CLAUDE.md                    # AI 튜터 행동 규칙
├── main.py                      # 학생 4 담당 — 메인 메뉴, 전체 흐름
├── quiz.py                      # 학생 1 담당 — 퀴즈 출제 & 정답 확인
├── data.py                      # 학생 2 담당 — 문제 데이터 관리
├── score.py                     # 학생 3 담당 — 점수 기록 & 조회
├── questions.json               # 퀴즈 문제 저장 파일
├── scores.txt                   # 점수 기록 파일
├── requirements.txt
└── tests/
    ├── test_quiz.py
    ├── test_data.py
    └── test_score.py
```

### 역할 분담

| 학생 | 파일 | 주요 학습 개념 |
|------|------|--------------|
| 1 | quiz.py | 함수, if문, 반복문 |
| 2 | data.py | 리스트, 딕셔너리, JSON 파일 읽기 |
| 3 | score.py | 파일 I/O, 문자열 처리 |
| 4 | main.py | 모듈 import, 전체 흐름 설계 |

### 실행 흐름

```
main.py 실행
    → 메인 메뉴 출력
    → 퀴즈 시작 선택
        → data.py: 문제 불러오기
        → quiz.py: 문제 출제 & 답 확인
        → score.py: 점수 저장
    → 기록 조회 선택
        → score.py: 저장된 기록 출력
```

## 4. 학습 단계

### 단계 0 — 첫 세션 (아이스브레이킹)

Git 명령어보다 먼저 **서로 알아가는 시간**입니다.

1. 자기소개 후 페어 프로그래밍으로 **일부러 망가진 코드 함께 고치기**
2. 고치면서 에러 메시지 읽는 법, 검색하는 법 자연스럽게 익히기
3. AI에게 질문하는 법 함께 연습

### 단계 1 — 환경 설정 & 첫 커밋

- Codespaces 열기, 저장소 확인
- 자기 이름으로 브랜치 만들기
- `hello.py` 작성 후 첫 커밋 & 푸시
- AI가 Git 명령어를 대신 실행하고 "지금 이런 상황에서 이런 명령을 했어요" 설명

### 단계 2 — 쉐도우 코드 채우기

- **코드 작성 전에 한국어로 먼저 쓰기**
  ```
  1. 이 함수가 할 일을 한국어 문장 3개로 쓰기
  2. 번호 목록으로 단계 나누기
  3. 그 다음에 Python으로 번역하기
  ```
- 로컬에서 `pytest` 실행해서 초록불(PASSED) 확인
- 통과하면 PR 올리기

### 단계 3 — 모듈 연결 & 전체 실행

- 4개 모듈을 `main.py`로 연결
- 전체 프로그램 실행해보기
- Pull Request → GitHub Actions 자동 테스트 → merge

### 단계 4 — 테스트 & 코드 리뷰

- 서로의 코드 읽고 PR 코멘트 남기기
- 테스트 케이스 추가 작성
- 매 세션 마지막 15분: 회고 ("오늘 뭘 배웠나?")

## 5. 검증 방식

### 로컬 테스트

```bash
pytest tests/test_quiz.py   # 내 모듈만 테스트
pytest                       # 전체 테스트
```

초록불(PASSED) = 성공, 빨간불(FAILED) = 다시 도전

### GitHub Actions CI

PR을 올리면 자동으로 테스트가 실행됩니다.  
모든 테스트가 통과해야 merge 가능합니다.

### 테스트 우선 구조

테스트 코드는 선생님이 미리 작성해둡니다. 학생은 테스트를 통과시키는 것이 목표입니다.

```python
# tests/test_quiz.py (미리 작성됨)
def test_correct_answer():
    result = check_answer("1", "1")
    assert result == True          # 이걸 통과시키면 됩니다!

def test_wrong_answer():
    result = check_answer("2", "1")
    assert result == False
```
